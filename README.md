C++ Practice Exercises

Collection of small C++ exercises covering structures, arrays, dynamic memory, functions, and basic OOP concepts.
vjezba_g1.cpp

    Structure: Student {name, grade}
    Global array of 50 elements
    Menu with options 1-4, 9
    Concepts: struct, array, input validation (grade 1-5), functions with/without return, dynamic allocation

vjezba_g2.cpp

    Structure: Rectangle {side_a, side_b, area, perimeter}
    Global array of 50 elements
    Menu with options 1-4, 9
    Concepts: struct, array, computed fields, dynamic allocation, input validation, functions with/without return

vjezba_g7.cpp

    Structure: BoolPair {value1, value2}
    Global array of 50 elements
    Menu with options 1-4, 9
    Concepts: struct, array, boolean input validation, counting, dynamic allocation, random number generation, functions with return

vjezba_9_zadatak_25.cpp

    Structure: Circle {name, radius, circumference, area}
    Array of 10 elements
    Menu with options 4-6
    Concepts: struct, array, calculation formulas, search by name
